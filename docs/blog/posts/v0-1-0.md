# Mimir v0.1.0 — the well opens

The first release of Mimir is here: a local-first second brain that embeds your
notes into a Qdrant vector store on your own machine and lets you query them as
semantic search or full RAG chat. Nothing leaves your Mac.

<!-- more -->

## What's in it

- **Local Qdrant vector store** via `docker-compose.yml` (REST on 6333, gRPC on
  6334).
- **Native Apple Silicon embeddings** through `fastembed` with
  `BAAI/bge-small-en-v1.5` (384d) as the default.
- **A single `mimir` CLI** (Typer) with `status`, `ingest`, `query`, `chat`,
  `reset`, `watch`, and `version`.
- **Incremental ingestion** &mdash; per-file SHA-256 change detection skips
  unchanged files automatically.
- **Streaming RAG chat** via a local Ollama model, with citations back to your
  notes.
- **Live re-indexing** with `mimir watch` (requires `fswatch`).
- **Markdown / text / PDF / RST / org loaders**, a paragraph-aware chunker, and
  pre-indexed payload fields for fast filtered search.
- **Configuration** from `MIMIR_*` environment variables and `.env`.

## Getting started

```bash
git clone https://github.com/LaurMost/mimir.git && cd mimir
uv sync
docker compose up -d
uv run mimir ingest
uv run mimir query "what did I conclude last quarter"
```

See the [Quickstart](../../getting-started/quickstart.md) for the full walkthrough.
